	웹은 서버에 페이지로 요청(HTTP Request) 할 때
	두가지 방식을 취한다.(GET방식, POST방식)
	
	QueryString을 포함한 GET 방식과
	주소줄에 요청 페이지만 있고 QueryString이 없는
	상태에서 폼의 데이터를 하나로 묶어 히든 처리 되면서
	요청되는 POST 방식이 있다.
	
	QueryString 이란?

	주소 중에서 ?를 포함한 변수=값&변수=값&...
	을 말한다. (? 뒤에 있는 모든 것.)
